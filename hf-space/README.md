# OmniVoice — Chuyển Văn Bản Thành Giọng Nói

Mô hình TTS zero-shot hỗ trợ **646+ ngôn ngữ**, bao gồm tiếng Việt.

## Tính năng
- **Sao chép giọng nói** — Clone bất kỳ giọng nói nào từ audio tham chiếu 3-10 giây
- **Thiết kế giọng nói** — Tạo giọng nói theo mô tả (giới tính, độ tuổi, accent...)
- **GPU T4 miễn phí** — Sử dụng ZeroGPU của HuggingFace

## API

Space này cũng cung cấp REST API để tích hợp vào ứng dụng khác.
Xem tab "API" ở cuối trang để biết chi tiết.
