## Cách Tạo ADR Mới

1. Lấy số NNNN = số lớn nhất hiện có + 1
2. Tạo file: `docs/decisions/NNNN-title.md`
3. Cấu trúc: Bối cảnh → Quyết định → Hệ quả → Alternatives considered
4. Cập nhật index này
5. Cập nhật CLAUDE.md nếu là quyết định quan trọng
