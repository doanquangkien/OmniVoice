## Cách Tạo Discussion Mới

1. Tạo file: `docs/discussions/YYYY-MM-DD-topic.md`
2. Trình bày: bối cảnh → options → pros/cons → recommendation
3. Hỏi user chốt
4. User chốt → tạo ADR → chuyển discussion xuống "Đã Chốt"
